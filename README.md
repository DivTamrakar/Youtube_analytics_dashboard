# Youtube_analytics_dashboard
A YouTube Analytics Dashboard built using Power BI to analyse channel performance, audience engagement, watch time, views, and subscriber growth using exported YouTube data.

üîé Project Overview

The YouTube Analytics Dashboard is an interactive data visualization project built using Power BI to analyze channel performance and audience insights.
This dashboard provides a comprehensive view of video performance, subscriber growth, watch time trends, engagement metrics, demographics, traffic sources, and device usage.
The goal of this project is to transform raw YouTube data into actionable insights for improving content strategy and channel growth.
